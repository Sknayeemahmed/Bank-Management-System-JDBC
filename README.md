# 🏦 Bank Management System (Java, JDBC, MySQL)

## 📌 Description

Developed a console-based Bank Management System using Java and JDBC following a monolithic architecture approach.

## 🚀 Features

* Implemented core banking functionalities including:

  * User Registration
  * Login
  * Account Creation
  * Deposit
  * Withdrawal
* Designed and integrated MySQL database for persistent data storage
* Used **PreparedStatement** for secure and efficient database operations
* Implemented business logic such as **balance validation during withdrawals**
* Maintained modular code structure using method-based separation
* Built a **transaction tracking system** to log all financial activities

## 🛠️ Technologies Used

* Java
* JDBC
* MySQL
* SQL
